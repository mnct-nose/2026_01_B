# 2026_01_B　都道府県別観光来訪者数の比較

## 概要

日本観光振興協会が公開している**デジタル観光統計オープンデータ**を用いて、複数の月における都道府県別観光来訪者数を比較・考察したWebページです。

## 調査対象期間

| 月 | 上位都道府県（1位） |
|----|-------------------|
| 2026年4月 | 東京都（約2,412万人） |
| 2026年1月 | 東京都（約2,288万人） |
| 2025年10月 | 東京都（約2,277万人） |
| 2025年8月 | 東京都（約2,239万人） |
